# Dozzle

Real-time log viewer for Docker containers.

## Configuration

- **URL:** [https://dozzle.yourdomain.com](https://dozzle.yourdomain.com)
- **Host:** 192.168.x.5 (VM 101)
- **Port:** 8888

## Deployment

Managed via GitOps. Push to `main` branch to trigger automatic deployment via webhook.

For manual deployment:
```bash
bash deploy.sh
```
