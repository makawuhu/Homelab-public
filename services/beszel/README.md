# Beszel Hub

Lightweight server monitoring tool.

- **URL:** https://beszel.yourdomain.com
- **Host:** VM 101 (`192.168.x.5`)
- **Port:** 8091 (internal 8090)

## Initial Setup

1. Access https://beszel.yourdomain.com.
2. Create your admin account.
3. Click "Add System" to get the agent configuration (public key).
4. Deploy agents on target hosts with the key.
